Talk

Python is not Always Slow

Benny Huang

We all know Python is slow. But in many cases it is not. Before blindly complaining the slowness of Python, we need to know when it is slow. There are also many cases it's totally OK to be slow, so before optimizing the code, we need know to when it's worth our time to rewrite our code. I will talk about Python's performance in different fields such as scientific computing, machine learning, plotting, network programming...etc. And also mention some experience about profiling and accelerating the code.
